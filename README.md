## Install

This module is available on [npm](https://www.npmjs.com/).

```BASH
npm install -g strong-pwgen-cli
# or execute it directly with npx (since npm v5.2)
npx strong-pwgen-cli
```

## Usage


```BASH
pwgen [options]
```

## Options

### length

Adjust the length of the password.

```
pwgen --length 20
# or
pwgen -l 20
```
